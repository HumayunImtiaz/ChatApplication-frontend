# Frontend - Chat Application 💬

React + Next.js + TypeScript + Tailwind CSS + Socket.io

## Folder Structure

```
frontend/
├── src/
│   ├── app/          📄 Pages & layout
│   ├── components/   🎨 React components
│   ├── hooks/        🪝 Custom hooks
│   ├── services/     🔗 API calls
│   ├── utils/        🛠️  Helpers
│   └── styles/       🎨 Global styles
├── public/           📸 Static files
└── package.json      📦 Dependencies
```

## Quick Start

```bash
npm install
npm run dev
```

Browser: `http://localhost:3000` 🚀
